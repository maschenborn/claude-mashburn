# mashburn Audit Skill

Deep audit for a specific category. More thorough than inspect — checks versions, analyzes configs, detects conflicts, suggests improvements.

## Depth Levels

- **inspect** = "is it there?" (binary check)
- **audit** = "is it right?" (quality analysis)

When you find something worth capturing as a new practice, suggest it to Michael but never create it autonomously.
